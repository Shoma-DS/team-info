# Behaviors

*   **追従型（Floating）CTA:** 画面下部に常に「無料カウンセリングに参加する」の緑色のボタンが固定表示されています。
*   **スクロールアニメーション:** 各セクションが画面内に入ると、フェードインやスライドインで表示される演出が多用されています。
*   **スライダー:** 講師紹介、受講生実績、実践成果物のセクションで、横にスライド（Swiper等を利用）して情報を閲覧できるようになっています。
*   **アコーディオン:** FAQセクションはクリックで回答が開閉する仕組みです。
*   **ボタン:** 「簡単30秒で予約完了！」といったマイクロコピーを添えた、目立つ緑色のボタンが各所に配置されています。
*   **スティッキー:** ヘッダーの追従はなく、下部のCTAボタンのみがスティッキー（固定）設定されています。
