## このバッチの傾向

- `statement` フックが多数派
- パターンインタラプトはほぼ使わない
- 1カット主体で、素材は派手さより見やすさ重視
- BGM と SFX は語りを邪魔しない程度に補助

## 素材規格

- サイズ: 1080×1920px（縦型 9:16）
- 形式: `.jpg` / `.png` / `.mp4`（短いループ動画も可）
- 注意: 著作権フリー素材を使用してください
  - 推奨: Unsplash, Pexels, Pixabay, Wikimedia Commons
