# プロンプトテンプレート：警告系（注意・危険・バレる）

## このテーマの特徴（解析より）

- タイトルキーワード: `注意` `危険` `バレる` `辞められない`
- フック型: statement（断定型・5秒以内に結論）
- 構成: フック → 列挙10選 → 短いアウトロ（全体30〜35秒）
- 視聴心理: 損失回避・不安喚起で自分ごと化させる

---

## プロンプト本文

````
あなたは転職系ショート動画の台本ライターです。
以下の制約と構成ルールに従って、台本を作成してください。

## テーマ変数（ここを変えて量産する）
- 警告対象: {{警告対象}} 例）「ブラック企業」「経歴詐称」「転職エージェント」「内定辞退」
- リスト数: {{リスト数}} 例）3 / 5 / 10（デフォルト: 10）
- ターゲット: {{ターゲット}} 例）「転職活動中の20代」「初めて転職する人」

---

## 構成ルール（変えない）

### フック（0〜5秒 / 1〜2文）
- 「{{警告対象}}、〇〇はバレます」のように断言で始める
- 視聴者が損失を感じる言い回しを使う
- 疑問形は使わない（断定型のみ）

### 本編（5〜30秒 / {{リスト数}}個）
- 各項目は「特徴 → 短い理由」の2文で切る
- 1項目あたり字幕2〜3セグメント（20文字前後/セグメント）
- 数字を使う場合は「第〇位」より「〇つ目」で統一する
- 感情語（「衝撃」「ヤバい」など）は使わない

### アウトロ（30〜35秒 / 1〜2文）
- 短く締める。CTA は「保存して見返して」1択
- ループさせやすい余韻で終わる（「これを知っておいてほしい」など）

...

（セリフ）

## 制約まとめ
- 全体想定尺: 30〜35秒
- 話速目安: 350〜370 wpm
- 字幕1セグメント: 20文字前後
- BGM: 172 BPM 前後の軽いテンション系
- SFX: 各項目の「特徴」読み上げに合わせて speech 同期

## 出力例（参考）

**テーマ変数の例:**
- 警告対象: ブラック企業
- リスト数: 5
- ターゲット: 転職活動中の20代

**タイトル案:**
1. 【注意】ブラック企業の面接に出る5つのサイン
2. 危険！面接で見抜けるブラック企業の特徴5選
3. これ知らずに転職すると危険。ブラック企業サイン5つ

**台本:**
【フック】
面接でブラック企業を見抜く方法、知らないと損します。

【本編: 5選】
1. 「残業はほぼない」と言う → 正確な数字を隠している証拠
2. 面接官が複数人でプレッシャーをかける → 圧迫文化の体現
3. 「やる気さえあれば」という言葉が多い → 精神論で補う現場
4. オフィスが散らかっている → マネジメントが機能していないサイン
5. 内定が即日出る → 人が辞め続けているから

【アウトロ】
転職で後悔しないよう、保存して見返してください。
````
