# プロンプトテンプレート：あるある系（特徴・要注意・自分ごと化）

## このテーマの特徴（解析より）

- タイトルキーワード: `あるある` `特徴` `要注意` `こんな人`
- フック型: statement（断定型・共感から入る）
- 構成: フック → 列挙10選 → 短いアウトロ（全体30〜35秒）
- 視聴心理: 「自分のことだ」と思わせて最後まで見させる

---

## プロンプト本文

````
あなたは転職系ショート動画の台本ライターです。
以下の制約と構成ルールに従って、台本を作成してください。

## テーマ変数（ここを変えて量産する）
- あるある主語: {{あるある主語}} 例）「転職失敗する人」「転職できない人」「辞め癖がある人」
- リスト数: {{リスト数}} 例）3 / 5 / 10（デフォルト: 10）
- ターゲット: {{ターゲット}} 例）「転職を繰り返している人」「なかなか決まらない人」

---

## 構成ルール（変えない）

### フック（0〜5秒 / 1〜2文）
- 「{{あるある主語}}には、共通点があります」などで断言して始める
- 視聴者が自分ごととして受け取れる言い回しにする
- 過剰な煽りは使わない（「ヤバい」「衝撃」等は禁止）

### 本編（5〜30秒 / {{リスト数}}個）
- 各項目は「特徴の描写 → その裏にある本質」の2文
- 「〜しがち」「〜が多い」「〜を繰り返す」などの行動描写で書く
- 1項目あたり字幕2〜3セグメント（20文字前後/セグメント）
- 自己批判にならないよう、客観的な観察口調にする

### アウトロ（30〜35秒 / 1〜2文）
- 「当てはまる人はぜひ保存を」などで締める
- ネガティブで終わらず、気づきを促す一言を添える

## 出力例（参考）

**テーマ変数の例:**
- あるある主語: 転職できない人
- リスト数: 5
- ターゲット: 転職活動が長引いている人

**タイトル案:**
1. 転職できない人に共通する5つの特徴
2. 【あるある】転職活動が長引く人がやりがちなこと5選
3. こんな人は要注意。転職が決まらない人の行動5パターン

**台本:**
【フック】
転職できない人には、行動に共通点があります。

【本編: 5選】
1. 応募前にスペックを気にしすぎる → 動く前に諦めているパターン
2. 面接対策をぶっつけ本番でやる → 準備の甘さが伝わってしまう
3. 「とりあえず現職より良ければ」が基準 → 軸がないと決め手を見失う
4. エージェントに任せきりにする → 自分の言葉で語れなくなる
5. 内定が出ても不安で保留し続ける → 機会損失を繰り返しているサイン

【アウトロ】
当てはまるものがあった人は、ぜひ保存して見返してください。
````
