## 運用メモ

- この回は字幕が主役なので、背景は情報量を増やしすぎない
- 同じ人物を追う必要はなく、論点ごとにイラストを切り替える
- ポップ寄りでも、色がうるさすぎるものは避ける
- タイトル回と CTA は、文字が乗っても読める余白があるものを優先する

## 優先して探したい論点

- 仕事前から疲れている
- 通知や上司に気が重くなる
- 評価されず諦める
- 将来が見えない
- 休日が回復だけで終わる
- 転職を考え始める
