## 素材規格

- **サイズ**: 1080×1920px（縦型 9:16）
- **形式**: `.jpg` / `.png`（短い動画ループ `.mp4` も可）
- **注意**: 著作権フリー素材を使用してください

### 推奨素材サイト
| サイト | 特徴 |
|---|---|
| [Unsplash](https://unsplash.com) | 高品質・完全無料 |
| [Pexels](https://pexels.com) | 動画素材も豊富 |
| [Pixabay](https://pixabay.com) | 日本語検索対応 |

### 検索キーワードのコツ（英語推奨）
- フック・CTA: `minimal dark background`, `abstract neon light`
- 人物系セクション: `portrait studio beautiful woman`, `athlete close up`
- 芸能系イメージ: `stage spotlight`, `red carpet`, `celebrity interview`
