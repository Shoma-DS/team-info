## 6. 台本構成パターン

- **基本形**: 「感情ワードタイトル ＋ 9選列挙 ＋ 共感締め」
- **各項目の構造**:
  1. 見出し（例: 「1 会社への口が減った」）
  2. 状況説明（「〜という人は…」「以前は〜だったのに…」）
  3. 心理解釈（「これは〜という心理からきている」）
  4. 共感補強（「そういう人は〜なのです」）
- **文体**: 第三者観察型・共感誘発型（断言より「そういう人は〜」）
- **特徴的な言い回し**:
  - 「以前は〜だったのに今は〜」の対比構造
  - 「なぜかというと〜」「というのは〜」で理由を補足
  - 「〜なのです」「〜なのかもしれません」の語尾で押しつけない温度感

## 9. バズ要因まとめ

- **テーマの共感軸が強い**: 「辞める一歩手前」「もう無理」「心が出社していない」など、現在進行形の不安に直接刺さる
- **観察型の文体**: 「自分のことを言われている」感を煽らず、でも当てはまってしまう設計
- **9選の網羅感**: 途中で自分に当てはまる項目があると最後まで見てしまう
- **長尺でも成立する理由**: 各項目が独立した「あるある」として完結しており、飽きずに積み上がる
- **BGMを絞ることで集中を誘導**: 音が少ない分、語りに引き込まれやすい

## 10. 推奨クリエイティブ方針

- タイトルは **【感情/状態ワード】＋ 〇〇の特徴9選 / 理由9選** の形を維持する
- 感情ワードはネガティブな状態を肯定的に言語化する（「逃げたい」ではなく「ここじゃない感」）
- 冒頭はタイトルを読み上げて即番号コール。余計な前置きは不要
- 各項目は **見出し → 状況説明 → 心理解釈 → 共感補強** の4ステップで60〜90秒
- 文体は第三者観察型。断言より「〜という人は」「〜なのです」の温度感を保つ
- 字幕は白字＋黒ボックス（opacity 0.92〜0.95）。縦ショートの plain_light とは別スタイル
- BGMは123 BPM 程度を薄く。SFXは最小限
- 背景は1枚固定でも成立する。素材コストが低い

### 注意点

- このバッチは横動画（16:9）。縦ショート（9:16）とは字幕サイズ・配置・レンダリング設定が別物
- `career-listicle` profile の `remotion_mode: image-crossfade` は縦用の設定。横動画用には別途調整が必要
- 尺が10〜15分あるため、Remotion のレンダリング時間も縦の10〜20倍かかる見込み
