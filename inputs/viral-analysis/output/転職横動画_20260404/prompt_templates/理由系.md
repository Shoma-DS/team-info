# プロンプトテンプレート：理由系（賢い人が〜しない理由 / できない理由）

## このテーマの特徴（解析より）

- タイトルキーワード: 人物修飾ワード（`賢いひと` `わかっている人` `冷静な人`）＋ 〇〇しない/乗らない理由9選
- フック型: statement（「〜には理由があります」断言型）
- 構成: タイトルコール → 9つの理由列挙（各60〜90秒）→ 「この視点を持つと変わる」締め（全体8〜14分）
- 視聴心理: 「自分が悪いんじゃない」という安心感と「知っている人はもう動いている」という焦りの両立
- 文体: 理由・構造の解説型。「なぜかというと〜」「これは〜という原理」で論理的に展開
- バッチ代表作: 【今さら無理】賢いひとが会社の引き止めに乗らない理由9選

### タイトルコール（0〜15秒 / 2〜3文）
- タイトルをそのまま読み上げる

### 本編（15秒〜終盤 / {{リスト数}}個 / 1項目60〜90秒）
各項目は以下の4ステップで書く:
1. **見出し**（1文）: 「1 〇〇だから」のような理由タイトル
2. **状況説明**（2〜3文）: 「〜という言葉が出てきても」「〜と言われても」で提示される状況を描写
3. **理由解釈**（2〜3文）: 「なぜかというと〜」「この判断の背景には〜」で構造的に説明
4. **共感締め**（1〜2文）: 「だから〜なのです」「〜という判断が自然にできるようになっているのです」

- 「感情で動いていない」「構造的に理解している」という{{人物修飾ワード}}の視点を軸にする

- 「本来であれば〜のはず」「でも〜という現実がある」の対比を使う

1 職場は簡単に変わらないと知っているから
退職の引き止めで給料を上げる、役職をつけると言われることがあります。

賢い人は、言葉の中身より、その言葉が出たタイミングと文脈を見るのです。

【締め】
こういった視点を持てると、同じ引き止めの言葉を聞いても、判断が変わってきます。
