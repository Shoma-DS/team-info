# プロンプトテンプレート：方法系（見抜く方法・やり方・確認ポイント）

## このテーマの特徴（解析より）

- タイトルキーワード: 動詞ワード（`見抜く` `確認する` `判断する` `気づく`）＋ 〇〇の方法9選
- フック型: statement（「〜を見抜くポイントがあります」断言型）
- 構成: タイトルコール → 9つのポイント列挙（各60〜90秒）→ 「実際に使ってほしい」締め（全体8〜12分）
- 視聴心理: 「これを知っておけば自分で判断できる」という有益情報の獲得感
- 文体: 具体的な確認行動・観察ポイントを解説型で提示。「こういう場合は〜」「反対に〜なら〜」の対比

### 本編（15秒〜終盤 / {{リスト数}}個 / 1項目60〜90秒）
各項目は以下の4ステップで書く:
1. **見出し**（1文）: 「1 〇〇を確認する」のような行動タイトル
2. **具体的な観察ポイント**（2〜3文）: 「〜で聞いてみると分かります」「〜の反応を見てください」
3. **解釈の補足**（2〜3文）: 「こういう反応が返ってきたら〜」「反対に〜ならば〜」の分岐で説明

- 「知識の提供」ではなく「実際に使える行動レベル」に落とす

- 「こういう場合は注意」「こういう場合は安心」の対比を意識する

### 締め（最後 20〜30秒 / 2〜3文）

2. 【実は簡単】面接だけで分かるやばい会社のサイン9選

**台本（冒頭2項目のみ）:**

【タイトルコール】
面接でブラック企業を見抜く方法、9選です。
では、1つ目。

【本編】

1 面接官の言葉に「うちは」が多いか確認する
面接で自社の強みを話す時に、「うちはここが他と違う」「うちはこういう文化だから」と頻繁に言う会社があります。
自社の良さを伝えたいのは理解できますが、比較の軸が外部ではなく自社内だけになっている場合、外の基準で物事を見る習慣がない可能性があります。
なぜこれを確認するかというと、入社後に「うちでは普通」「他はどうか知らないけど」という感覚で理不尽なことが通っていく組織かどうかが見えてくるからです。

2 残業の話題を自分から出してみる
面接の終わりに「業務量や残業についてお聞きしてもいいですか」と聞いてみてください。
「それほど多くないです」という答えと「時期によりますが〜」という答えでは、答え方の丁寧さが全然違います。
曖昧にしか答えられない場合、正確な情報を把握していないか、言いづらい実態がある可能性があります。
この一言を聞くだけで、面接官が現場の実態を把握しているかどうかが見えてくるのです。

【締め】
全部完璧な会社はなかなかないですが、この観点を持っておくだけで、入社後の後悔を減らせます。
ぜひ保存して、転職活動に役立ててください。
