# テンプレート: acoustic_cover

## 用途
- 1曲カバーのリリックビデオ制作
- カラオケ風歌詞表示付きアコースティックカバー動画

## 基本設定
- 解像度: 1920x1080
- fps: 30
- 想定尺: 3-5分（楽曲に依存）

## シーン構成
1. イントロ（5-10秒）
   - 曲名をフェードイン表示
   - "Original by [アーティスト名]" 表示
   - "Covered by AcoRiel" 表示
   - 背景はメインと同じ（暗めフィルタ強め）

2. 歌パート（メイン）
   - 歌詞字幕はRemotionでは実装しない（後で手動編集）
   - 背景画像 + グラデーションオーバーレイ
   - 光のパーティクルエフェクト
   - 画面下部にオーディオスペクトラム

3. アウトロ（5-10秒）
   - 歌詞表示をフェードアウト
   - "AcoRiel" ロゴ/テキスト表示
   - チャンネル登録案内テキスト
   - フェードアウトで終了

## 編集要件
- 歌詞はLRCタイムスタンプで正確に同期させる
- 歌詞のフェードイン/フェードアウトと軽いぼかし遷移を滑らかにする
- 背景は低刺激（超低速パン/ズームのみ）
- パーティクルは視認できる量・サイズを確保しつつ、歌詞判読を阻害しない
- スペクトラムは小さく、補助的な位置に

## テキスト要件
- 歌詞は1行あたり最大30文字程度
- フォントサイズは視認性優先で大きめ（52-60px）
- 歌詞の行間は十分に取る

## 素材入力フロー（1つずつ確認）
選択時の共通ルール:
- 各項目は対象フォルダ内の実在ファイルを列挙して、番号付き選択肢として提示する。
- 拡張子条件に合うファイルのみ候補に出す。
- 候補が0件ならその項目は確定せず、不足素材として報告する。

1. カバー曲音源を選択
   - 選択元フォルダ: `Remotion/video_resources/channels/acoriel/assets/songs/`
   - ファイル形式: `wav` または `mp3`
   - 推奨: マスタリング済み、適切な音量レベル

2. 歌詞ファイルを選択（任意・スキップ可能）
   - 選択元フォルダ: `Remotion/scripts/lyrics/`
   - ファイル形式: `lrc`
   - 歌詞字幕は後で手動編集するため、基本スキップ

3. 背景画像を選択
   - 選択元フォルダ: `Remotion/video_resources/channels/acoriel/assets/backgrounds/`
   - ファイル形式: `jpg` または `png`
   - 推奨: 1920x1080以上、ダーク系

4. エフェクト設定を選択
   - デフォルト: 光の粒子パーティクル（シルバー/白系）
   - 既存テンプレート選択元: `Remotion/video_resources/channels/acoriel/effects/templates/`
   - 新規カスタム時: その都度、内容に合う日本語名を付けてテンプレート保存する
   - 保存内容: 効果の目的 / 適用シーン / 主要パラメータ / 禁止事項

## 実装仕様（Remotion）
- 背景画像: 動画全尺で表示、超低速パン/ズーム、brightness(0.4) saturate(0.7)
- 歌: タイムライン基準トラック（volume=1.0）
- 歌詞: 手動編集で後から追加（Remotionコード側では実装しない）
- スペクトラム: 画面下部に線形バー表示、シルバー/白系
- パーティクル: 光の粒子をやや多め・大きめ・ぼかし強めで浮遊させる
- イントロ/アウトロ: Sequenceで制御、フェードイン/アウト

## LRC歌詞フォーマット仕様
```
[ti:曲名]
[ar:Original Artist名]
[al:アルバム名]（オプション）
[by:AcoRiel]
[00:00.00]（前奏）
[00:15.50]最初の歌詞行
[00:22.00]二番目の歌詞行
...
```
- 標準LRC形式（[mm:ss.xx]形式）
- 空の歌詞行は間奏を示す
- メタデータタグは `[ti:]` `[ar:]` が必須

- [ ] アウトロにチャンネルロゴが表示される
- [ ] 歌詞の視認性が確保されている
- [ ] パーティクルが歌詞を邪魔していない
- [ ] スペクトラムが曲に同期している
- [ ] 全体のトーンがエレガントで統一されている
