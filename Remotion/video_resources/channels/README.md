# チャンネル別動画テンプレート

このフォルダには、YouTubeチャンネルごとの動画制作ルールとテンプレートを置きます。

## 構造
```text
Remotion/video_resources/channels/
└── <channel_id>/
    ├── channel_info.md
    └── templates/
        ├── <template_name>.md
        └── ...
```

## ルール
- `channel_info.md`: チャンネル固有ルール（トーン、禁止事項、CTA、デザイン方針）
- `templates/*.md`: 動画テンプレート（尺、シーン構成、編集要件、チェック項目）
