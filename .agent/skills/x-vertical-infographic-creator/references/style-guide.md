## 避けるカラー

- `#000000` 純黒背景
- `#FF0000` 赤系メインカラー
- 彩度が高すぎるネオン系
- グラデーションの多用（1箇所のみ可）

## フォント方針

| 要素 | スタイル |
|------|----------|
| 大見出し | 太字・最大サイズ・1行16文字以内 |
| サブ見出し | 中太字・大見出しの60〜70%サイズ |
| ステップラベル | 中太字・白カード内 |
| 補足テキスト | 細字・最小サイズ |
| 結論ブロック | 太字・大見出しに次ぐサイズ |

日本語テキストのルール：
- 1行あたり **10〜16文字** 以内
- 長い文章は改行して分割する
- 読んで2秒以内に意味が取れる密度にする

## レイアウト

### 縦型 9:16（基本）

```
┌──────────────────────┐  ← 上部 (約30%)
│ 大見出し              │
│ サブ見出し            │
│                [キャラ]│
├──────────────────────┤  ← 中央 (約45%)
│ ┌────┐ → ┌────┐      │
│ │ 1  │   │ 2  │      │
│ └────┘   └────┘      │
│ ┌────┐ → ┌────┐      │
│ │ 3  │   │ 4  │      │
│ └────┘   └────┘      │
├──────────────────────┤  ← 下部 (約25%)
│ 結論（大きく）        │
│ 補足（小さく）        │
└──────────────────────┘
```

### コンテンツ密度ルール

| 良い例 | 悪い例 |
|--------|--------|
| 3〜4ステップのみ | 8項目の箇条書き |
| 大見出し2行以内 | 説明文を画像内に全部入れる |
| 結論1文 | 複数の結論・CTA |
| アクセント1色 | 3色以上の強調 |

## キャラクター配置

詳細は `../accounts/gutaraaikatuyou/character-prompt.md` を参照。

共通ルール：
- サイズ: 図全体の **20〜25%** 以内
- 位置: 上部右端 / 右下 / 結論横（吹き出し風）
- キャラを見出しや流れの上に被せない
- キャラより情報設計が優先
