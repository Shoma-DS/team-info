# 改善プロンプト集 — x-vertical-infographic-creator

生成画像が期待と違う場合に使う修正指示テンプレート。
状況に合わせて1つ以上を組み合わせる。

---

## 雰囲気・色調の問題

### 怪しい・情報商材感が出た

```
黒背景と赤・オレンジ系の煽り表現をすべて取り除いてください。
背景を白〜淡いブルー (#EAF6FF〜#F5FBFF) に変更し、
信頼感のあるSaaS・AIサービス紹介のような爽やかな印象にしてください。
ギラギラしたネオン、グロー効果、過剰な光沢も削除してください。
```

### 暗すぎる・重い雰囲気

```
全体の配色を明るく軽くしてください。
背景: #EAF6FF（薄い水色）、テキスト: #0F172A（ネイビー）を基本にし、
ダークモードや深色系の要素を削除してください。
清潔感のある昼間のオフィス空間のような印象を目指してください。
```

### 地味すぎる・会社資料感が強い

```
全体はシンプルに保ちながら、メインブルー (#2563EB) のカードボーダーと
シアン (#38BDF8) の矢印・アイコンを追加してアクセントを付けてください。
最重要ワードにだけイエロー (#FACC15) のハイライトを入れてください。
「テック感」と「親しみやすさ」のバランスを取った仕上がりにしてください。
```

---

## 情報量の問題

### 情報が多すぎる・読みにくい

```
文字量を30〜40%削減してください。
中央のフロー（3〜4ステップ）と下部の結論だけが
一瞬で読める構成にしてください。
細かい説明文、長い箇条書き、補足テキストはすべて削除してください。
1枚で「何が変わったか」「次に何をするか」だけが伝わればOKです。
```

### 項目が多すぎる（9項目以上）

```
現在の項目を最大7個に絞ってください。
最も伝えたい3〜4ステップのフローのみを残し、
残りは投稿文（テキスト側）に任せる設計に変えてください。
```

---

## テキストの問題

### 文字が崩れた・日本語が読めない

```
日本語テキストをすべて短く分解し直してください。
1行あたり最大16文字以内、大見出しは2〜3行に収めてください。
画像内に入れる文字は最小限にし、小さい補足テキストをなくしてください。
太字・大きいフォントのみで構成するとバランスが良くなります。
```

### 文字が小さすぎる

```
全体のフォントサイズを20〜30%大きくしてください。
特に大見出しと結論ブロックのテキストを最大サイズにしてください。
スマホで縮小表示されても読めるサイズ感を基準にしてください。
```

---

## キャラクターの問題

### キャラクターが目立ちすぎる・邪魔

```
キャラクターを補助要素として扱い、
上部右端または右下に小さく（図全体の20〜25%以内）配置し直してください。
キャラクターが見出しやフローカードの上に重ならないようにしてください。
情報設計の可読性を最優先にしてください。
```

### キャラクターが元気すぎる・別人になった

```
キャラクターの表情を「だるそう・半目・頬に手・舌少し出し」に戻してください。
元気な笑顔や真剣な顔にしないでください。
スーツ（ダークグレー）と赤いネクタイを維持してください。
黒髪・ぼさぼさ短髪のアニメ2Dイラスト調を維持してください。
```

---

## レイアウトの問題

### 上部・中央・下部の区分けが崩れた

```
縦型レイアウトを3ゾーンに再構成してください:
- 上部（全体の約30%）: 大見出し + サブ見出し + キャラクター
- 中央（全体の約45%）: 3〜4ステップの白カードフロー
- 下部（全体の約25%）: 結論（大きく） + 補足（小さく）
各ゾーンの間に適切な余白を取ってください。
```

### 横向き・正方形になった

```
縦型 9:16 比率で再生成してください（スマホ縦画面・X投稿最適化）。
```

---

## 組み合わせ例

「怪しくて情報過多だった場合」:
1. 「怪しい・情報商材感が出た」の修正を適用
2. 「情報が多すぎる」の修正を適用
3. 仕上げに「地味すぎる」の修正でアクセントを追加
