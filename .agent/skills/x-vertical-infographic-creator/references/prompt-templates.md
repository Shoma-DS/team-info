```

- {従来の手間2}
- {従来の手間3}

- {改善点1}
- {改善点2}
- {改善点3}

【下部】
```

## 改善が必要なとき

生成画像が期待と違う場合は `improvement-prompts.md` を参照する。

## 使い方

1. 下書きのテーマに合うテンプレートを選ぶ
2. `{}` の部分を実際の内容に置き換える
3. `../accounts/{account}/character-prompt.md` のキャラクター情報を追記する
4. 完成プロンプトを画像生成AIに渡す
