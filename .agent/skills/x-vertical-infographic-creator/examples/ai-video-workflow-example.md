# 完成例: AI動画制作ワークフロー図解

このファイルは「Claude Code × Remotion × Codex で動画制作を自動化」というテーマの
完成した図解プロンプト例です。Codex/Claude がプロンプトを生成するときの参考にする。

---

## 入力（下書きの要約）

- テーマ: AI動画制作ワークフローの変化
- 投稿の核心: アイデアを言うだけでRemotionが動画を作る時代になった
- ターゲット: AI初心者〜副業探し層〜AI量産中級者
- キーワード: Claude Code, Remotion, Codex, 自動化

---

## 図解コンセプト

- **何を一瞬で伝えるか**: 動画制作の手間がAIでゼロになった
- **誰に刺すか**: 「動画を作りたいけど難しそう」と思っている副業層
- **起こしたい感情**: 「これなら自分もできそう」「やってみたい」

## レイアウト案

```

[中央フロー]

② コード生成

[下部]

```

## 完成プロンプト（そのまま使える）

```
縦型9:16インフォグラフィック。スマホのXフィード閲覧前提。

【テーマ】
Claude Code・Remotion・Codexを使ったAI動画制作ワークフローの変化

【雰囲気】
爽やかなブルー基調、清潔感のあるSaaS風、信頼感と未来感を両立。
黒背景・赤文字・ネオン・情報商材感は一切禁止。

【上部】
大見出し（太字・2行・各16文字以内）:
「動画制作の前提が変わった」

サブ見出し（中太字・1行）:
「Claude Code × Remotion × Codex」

キャラクター配置:
ぐーたらAI社長を上部右端に配置（図の20%以内）。
外見: 黒髪ぼさぼさ短髪・半目・右頬に手・舌少し出し・ダークスーツ・赤ネクタイ。
アニメ2Dイラスト調。だるそうだが知性を感じる表情を維持。

【中央フロー】
白いカード4枚をブルーの矢印（→）でつなぐ:
カードボーダー: #2563EB

① 「アイデアを言う」（アイコン: 吹き出し）
② 「コード生成」（アイコン: コード記号）
③ 「Remotionで完成」（アイコン: 動画・フィルム）

「作業ゲーから」

【カラー詳細】

大見出し: #1E3A8A（ディープブルー）
カードテキスト: #0F172A（ネイビー）
カードボーダー: #2563EB（メインブルー）
矢印: #38BDF8（シアン）
強調1点: #FACC15（イエロー、「設計ゲー」のみ）

【NG指定】
黒背景禁止・赤テキスト禁止・ネオン禁止・グロー禁止・情報過多禁止。
```

## 改善プロンプト（問題が出た場合）

**怪しい感じになった場合**:
```
黒背景と赤・オレンジ系をすべて取り除き、背景を#EAF6FFに変更してください。
SaaS・AIサービスの清潔な印象にしてください。
```

**情報が多すぎる場合**:
```
4ステップのフローと下部の結論だけを残し、他の説明をすべて削除してください。
```

**キャラが目立ちすぎる場合**:
```
キャラクターを上部右端に小さく（20%以内）配置し直し、
見出しとフローを邪魔しないサイズにしてください。
```
