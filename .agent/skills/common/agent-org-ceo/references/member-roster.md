# Member Roster

## CEO
- 役割: オーナー依頼の再定義、優先順位づけ、担当アサイン、最終統合
- 読むもの:
  - `.agent/skills/skill-finder/SKILL.md`
  - `references/operating-model.md`
- 読まないもの:
  - 各スキル本文を最初から全部
- 主な判断:
  - どのメンバーに振るか
  - 何を先にやるか
  - どこでユーザー確認を挟むか

## Ops Manager
- 役割: Git、セットアップ、repo 整理、共有資産、引き継ぎ
- 主担当スキル:

  - `team-info-setup`
  - `shared-agent-assets`
  - `repo-information-architecture`
  - `context-handoff`

## 役割追加ルール
- 既存メンバーの責務が広すぎるときだけ追加する
- 新メンバーを増やす前に、既存メンバーへ寄せられないか確認する
- 新メンバーを追加したら、この表と `skill-finder` の該当箇所を更新する
