# 台本フォーマット定義

このドキュメントでは、台本作成時に選択できる「短尺」と「長尺」の詳細な定義を記載します。

---

## 短尺（5-15分）

### 基本情報
- **推奨時間**: 5-15分
- **用途**: SNS動画、YouTube短編、教育コンテンツ、プレゼンテーション
- **プラットフォーム**: YouTube Shorts、Instagram、TikTok、Twitter/X、教育プラットフォーム

### 構成

#### 全体構造
```
1. オープニング（30秒-1分）
2. 本編（4-12分）
   - セクション1（2-4分）
   - セクション2（2-4分）
   - セクション3（2-4分）※オプション
3. エンディング（30秒-1分）
```

#### オープニング（30秒-1分）
- **挨拶**: 簡潔に（5-10秒）
- **フック**: 「今日は○○について話します」「この3つを知っておくと○○」（10-20秒）
- **テーマ紹介**: 今日のトピックを明確に（10-20秒）
- **CTA（オプション）**: 「最後まで見てね」「いいね・チャンネル登録よろしく」（5-10秒）

#### 本編（4-12分）
- **セクション数**: 1-3つ
- **各セクションの長さ**: 2-4分
- **構成**:
  - ポイント提示 → 説明 → 具体例 → まとめ
  - または: 問題提起 → 解決策 → ベネフィット

#### エンディング（30秒-1分）
- **まとめ**: 今日のポイントを3つ以内で復習（10-20秒）
- **CTA**: いいね・チャンネル登録・コメント促進（10-20秒）
- **次回予告**（オプション）: 次回のテーマを軽く紹介（5-10秒）
- **締めの挨拶**: 「ではまた！」など（5秒）

### 特徴

#### テンポ
- **速い**: 情報を凝縮し、無駄を削ぎ落とす
- **リズミカル**: 短文を重ねてリズムを作る
- **メリハリ**: 強調ポイントを明確に

#### 文体
- **短文中心**: 1文15-30文字程度
- **簡潔**: 「つまり」「要するに」で要点を明確に
- **口語的**: 話しかけるような自然な言葉遣い
- **カジュアル**: 固すぎない表現（アカウントによる）

#### フック
- **冒頭3秒が勝負**: 視聴者の興味を即座に引く
- **疑問形**: 「○○って知ってますか？」
- **数字**: 「3つの理由」「5つのコツ」
- **意外性**: 「実は○○なんです」

#### 情報量
- **1つのメインテーマ**: 1本の動画に1つの大きなテーマ
- **3つ以内のポイント**: 覚えやすい数に絞る
- **深掘りしすぎない**: 表面的でも全体像を掴める程度

### 適したコンテンツ
- ハウツー（「○○の方法」）
- リスト形式（「5つの○○」）
- クイックティップス
- ニュース解説
- 商品レビュー
- Before/After紹介

### 推奨文字数
- **全体**: 1,500-4,500文字
- **1分あたり**: 約300文字（朗読速度による）

### 例: 短尺台本のサンプル構成

```
【オープニング】30秒
こんにちは、○○です。
今日は「北欧神話の神々3選」を紹介します。
意外と知らない北欧神話の面白い神々を見ていきましょう！

【本編】8分
■第1の神: オーディン（2.5分）
- 特徴: 片目の最高神
- エピソード: 知恵のために目を捧げた
- 現代への影響: 水曜日の語源

■第2の神: トール（2.5分）
- 特徴: 雷神、ハンマーを持つ
- エピソード: 巨人との戦い
- 現代への影響: マーベル映画

■第3の神: ロキ（3分）
- 特徴: いたずら好きのトリックスター
- エピソード: 変身の達人
- 現代への影響: 複雑なキャラクターの原型

【エンディング】30秒
今日は北欧神話の3柱の神を紹介しました。
気に入ったらいいね・チャンネル登録お願いします！
次回は「ギリシャ神話 vs 北欧神話」です。お楽しみに！
```

## 長尺（40-120分）

### 基本情報
- **推奨時間**: 40-120分
- **用途**: 睡眠導入、長編朗読、深い教育コンテンツ、ポッドキャスト
- **プラットフォーム**: YouTube（本編動画）、Podcast、音声配信

### 構成

#### 全体構造（睡眠導入型）
```
1. オープニング（2-3分）
2. 本編（40-50分）
   - 第1話（10-15分）
   - 第2話（10-15分）
   - 第3話（10-15分）
   - 第4話（10-15分）
3. 繰り返し部分（40-50分）
   ※本編と同じ内容を2周目として読む
4. エンディング（BGMのみ）
```

#### オープニング（2-3分）
- **挨拶とチャンネル紹介**: 「○○のゆです。いつもご視聴いただきありがとうございます」（20-30秒）
- **チャンネルの目的説明**: 「このチャンネルは睡眠用の朗読チャンネルです」（10-20秒）
- **聴き方の案内**: 「お休みになりながら〜」（10-20秒）
- **タイムスタンプ案内**: 「概要欄のタイムスタンプをご利用ください」（10-20秒）
- **今回のテーマ紹介**: 今日の内容の概要（30-60秒）
- **シリーズ説明**（該当する場合）: 全体の構成（20-30秒）
- **チャンネル登録促進**: 「お休みになる前に〜」（20-30秒）
- **おやすみの挨拶**: 「それではゆっくりおやすみなさい」（5-10秒）

#### 本編（40-50分）
- **章タイトル**: 「第○章 [メインタイトル]」
- **セクション数**: 3-4つ
- **各セクションの長さ**: 10-15分
- **構成**:
  - 語り出し → 背景説明 → 本題 → 情景描写 → 展開 → 締めくくり

#### 繰り返し部分（40-50分）
- **本編と同じ内容**を2周目として読む
- **特別な案内なし**: シームレスに繰り返しに入る
- **目的**: 睡眠導入効果を高める、1周目で寝落ちしなかった人も2周目で眠れる

#### エンディング
- **BGMのみ**: 静かに終わる
- **視聴者は自然に眠りに落ちる**

### 特徴

#### テンポ
- **ゆったり**: 急がず、落ち着いたペース
- **繰り返し**: 同じリズムパターンを繰り返す
- **催眠効果**: 聞き手を眠りに誘う

#### 文体
- **中文中心**: 1文30-60文字程度
- **丁寧**: 「〜でございます」「〜でした」など
- **詩的**: 情景描写が豊か
- **古風**（オプション）: 神話などでは古風な言い回しも

#### リズムと句読点
- **句読点を多用**: 対句表現、並列表現
- **繰り返しのパターン**: 同じ構造を繰り返す
- **音読しやすさ**: 朗読を前提とした文章

#### 情景描写
- **五感に訴える**: 視覚、聴覚、触覚、嗅覚、味覚
- **色と光**: 「青い空」「柔らかな光」
- **音と静寂**: 「風の囁き」「静かな夜」
- **時間の流れ**: 「やがて」「ほどなくして」

#### 知識の提示
- **物語として**: 説明的にならない
- **自然に織り込む**: 専門用語は使った後に説明
- **教育と娯楽のバランス**: 知識を楽しく

### 適したコンテンツ
- 神話（ギリシャ、北欧、日本、エジプトなど）
- 宇宙の旅
- 古代文明
- 美術史
- 哲学思想
- 自然科学（海洋、植物、鉱物など）
- 歴史物語

### 推奨文字数
- **全体（1周分）**: 12,000-15,000文字
- **繰り返し込み**: 24,000-30,000文字
- **1分あたり**: 約250-300文字（ゆっくりめの朗読）

### 例: 長尺台本のサンプル構成

```
【オープニング】3分（約900文字）
寝ながらトラベルのゆです。いつもご視聴いただきありがとうございます。
このチャンネルは睡眠用の朗読チャンネルです。
（中略）
それではゆっくりおやすみなさい。

【本編】45分
■第1章: 北欧神話の世界

第1話: ギンヌンガガップ（12分、約3,000文字）
はるかなる昔、時の名前すら持たぬ遠き時代のこと。
世界には、まだ何もございませんでした。
（中略）

第2話: ユミルと霜の巨人（11分、約2,750文字）
ギンヌンガガップの中心、氷と炎が出会う場所で...
（中略）

第3話: オーディンと世界の創造（11分、約2,750文字）
オーディン、ヴィリ、ヴェ。三人の兄弟は...
（中略）

第4話: 世界樹ユグドラシル（11分、約2,750文字）
全ての世界の中心に、一本の巨大な木が...
（中略）

【繰り返し部分】45分
（本編と同じ内容を再度読む）

【エンディング】
（BGMのみ）
```

## 比較表

| 項目 | 短尺（5-15分） | 長尺（40-120分） |
|------|---------------|------------------|
| **時間** | 5-15分 | 40-120分 |
| **セクション数** | 1-3つ | 3-4つ |
| **オープニング** | 30秒-1分 | 2-3分 |
| **本編** | 4-12分 | 40-50分 |
| **繰り返し** | なし | あり（2周） |
| **エンディング** | 30秒-1分 | BGMのみ |
| **文の長さ** | 15-30文字（短文） | 30-60文字（中文） |
| **テンポ** | 速い | ゆったり |
| **文体** | 簡潔、口語的 | 丁寧、詩的 |
| **情報量** | 凝縮 | 深掘り |
| **用途** | SNS、教育 | 睡眠導入、朗読 |
| **文字数（全体）** | 1,500-4,500文字 | 24,000-30,000文字 |

## 選択ガイド

### 短尺を選ぶべき場合
- ✅ SNSで拡散したい
- ✅ 忙しい人向けに情報を届けたい
- ✅ テンポよく複数のトピックを扱いたい
- ✅ ハウツーやティップスを紹介したい
- ✅ 視聴者の集中力を最後まで保ちたい

### 長尺を選ぶべき場合
- ✅ 睡眠導入を目的とする
- ✅ 深く掘り下げたいテーマがある
- ✅ 物語性のあるコンテンツ
- ✅ リラックス効果を重視
- ✅ 繰り返し聴いてもらいたい

---

## 注意事項

### 短尺の落とし穴
- ❌ 情報を詰め込みすぎる: 1つのテーマに絞る
- ❌ フックが弱い: 冒頭3秒で掴む
- ❌ テンポが遅い: 無駄な言葉を削る

### 長尺の落とし穴
- ❌ 説明的すぎる: 物語として語る
- ❌ テンポが速い: ゆっくり、落ち着いて
- ❌ 情景描写が少ない: 五感に訴える描写を
