# GSAP Awwwards Website Skill

## Goal
- Deliver a high-impact animated landing page using GSAP and ScrollTrigger.

## Workflow
1. Keep animation logic readable and section-scoped.
2. Implement layout with React 19 + Tailwind CSS 4 patterns in this project.
3. Tune scroll timing and easing to match content pacing.
4. Validate on desktop and mobile breakpoints.
5. Run lint/build and report any regressions.

## Guardrails
- Avoid over-animating critical reading areas.
- Prefer deterministic animation triggers over ad-hoc timers.
- Preserve existing brand direction unless explicitly changed.
