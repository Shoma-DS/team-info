# 候補者プロファイル抽出プロンプト

## 指示

以下の採用面接の文字起こしを読み、販売台本作成に必要な候補者プロファイルを抽出してください。

**抽出する情報**:

```
## 候補者プロファイル

### 基本情報
- 氏名・呼び方:
- 名前の揺れや誤認識がある場合の補足:
- 家族構成・育児状況:

### キャリア
- 直近の職歴（直近順）:
- 前職月収:
- 希望月収:
- 現在の働き方・シフト制約:

### AIスキル・学習意欲
- AIスキルレベル（ゼロ / 初心者 / 中級 / 上級）:
- PCスキルレベル:
- スクール検討歴:
- 学習意欲（根拠となる発言を引用）:

### 在宅ワーク志望の背景
- 在宅を希望する理由:
- 今の働き方で困っていること:
- 時間の制約（育児・介護・通院など）:

### 面接中の反応
- 興味を示したポイント（発言を引用）:
- やってみたい度（N/10点）:
- 印象的な発言・感情的な反応:
- 温度感（低 / 中 / 高）:

### 価値観・将来像
- 将来どうなりたいか:
- 憧れ・価値観の原体験:
- 刺さったビジョンや世界観:

### 痛点（解決したいこと・困っていること）
1.
2.
3.

### 販売上の整理
- 最大のフック:
- 提案しやすい切り口:
- 提案を難しくする要因:
- クロージング時に安心材料として先回りすべき点:
```

**ルール**:
- 文字起こしから読み取れる情報のみ記載する
- 推測は「（推測）」と明記する
- 発言の引用は「」で囲む
- 相手の価値観を決めた原体験や人物像が出てきたら、必ず拾う
- 名前に揺れがある場合は、動画タイトル・ユーザー確認・文字起こしのどこから採ったかを短く補足する
