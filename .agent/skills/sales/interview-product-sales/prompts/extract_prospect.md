# 候補者プロファイル抽出プロンプト

## 指示

以下の採用面接の文字起こしを読み、候補者プロファイルを抽出してください。

**抽出する情報**:

```
## 候補者プロファイル

### 基本情報
- 氏名・呼び方:
- 家族構成・育児状況:

### キャリア
- 直近の職歴（直近順）:
- 前職月収:
- 希望月収:

### AIスキル・学習意欲
- AIスキルレベル（ゼロ / 初心者 / 中級 / 上級）:
- スクール検討歴:
- 学習意欲（文字起こしから根拠となる発言を引用）:

### 在宅ワーク志望の背景
- 在宅を希望する理由:
- 時間の制約（育児・介護など）:

### 面接中の反応
- 興味を示したポイント（発言を引用）:
- やってみたい度（N/10点）:
- 印象的な発言・感情的な反応:

### 痛点（解決したいこと・困っていること）
1.
2.
3.

### 販売上の注意点
- 提案しやすいフック:
- 提案を難しくする要因:
```

**ルール**:
- 文字起こしから読み取れる情報のみ記載する
- 推測は「（推測）」と明記する
- 発言の引用は「」で囲む
