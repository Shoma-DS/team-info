# ベースクロージングフロー
## 出典: 営業ロープレフォーマット.pdf
## 用途: 商品固有の pitch_structure.md を作るときのベース

---

## 概要

採用面接・個別面談の場で商品を販売するための全フロー。
フェーズ1〜8を順番通りに進める。テストクロージング（フェーズ6）を飛ばして
クロージング（フェーズ7）に入らないこと。

---

## フロー一覧

| フェーズ | 名称 | 目的 |
|---------|------|------|
| 0 | 入室・ウォームアップ | 緊張をほぐす・今日の流れを予告する |
| 1 | アイスブレイク | ラポール構築・本日の目的確認 |
| 2 | きっかけのヒアリング | 応募経緯・現状・夢を引き出す |
| 3 | 目的の明確化 | なぜ変わりたいのか・何のためかを深掘り |
| 4 | コミットと目標設定 | 具体的な数字目標とコミット確認 |
| 5 | フルコミット確認 | 「本気か？」を問い直す |
| 6 | テストクロージング | 「合格」宣言・一緒にやる宣言 |
| 7 | クロージング | 商品説明・価格提示・申し込み |
| 8 | 反論処理 | 懸念点の解消・前進 |

---

## フェーズ0: 入室・ウォームアップ

**目的**: 候補者の緊張をほぐし、「今日は面談であってテストじゃない」という安心感を作る。
いきなり目的・ヒアリングに入らず、まず場の空気を温める。

**ポイント**:
- 接続・体調など当たり障りのない話題から入る
- 候補者の近況を一言聞いて「覚えていてくれた」という安心感を作る
- 今日の流れをざっくり予告して「何をされるかわからない不安」を消す
- 所要時間の目安を伝えておく

**セリフひな形:**
```
（接続確認）
「繋がりましたね。今日もよろしくお願いします。」

（近況の一言）
「[前回話した近況トピック（育児・体調など）]、その後どうですか？」

（今日の流れを予告）
「今日なんですけど、[所要時間]くらいで進めていこうと思っています。
前回お聞きしたこととかぶる質問をすることもあるかもしれないんですけど、
より深く知りたいなというところもあるので、もし同じ質問が出てきても
気にせず答えてもらえたら嬉しいです。
ざっくりそんな感じで進めていくので、気楽にいきましょう。
何か聞きたいことや気になることがあれば、途中で言ってもらって全然大丈夫です。」

（一呼吸おいてから次へ）
「じゃあ始めましょうか。」
```

## フェーズ1: アイスブレイク

```
「今日は前回お話しさせていただいた上での今回の面談なんですが、改めて、弊社の業務をなぜしたいと思ったか、お聞きしてもよろしいでしょうか？」

「今まで[候補者名]さんとやり取りをしてきて、[候補者名]さんの夢を今後実現させていく
環境を手にする為にここに来られたんですね。それを実現できるようにしていく為に
僕も全力で応援しますし、ただ[候補者名]さんはその環境を手にする為の面談のわけなので
本気で全力で後悔のないようにしてくださいね。」
```

## フェーズ2: きっかけのヒアリング

**質問リスト（順番通りに使う）:**
```
・それでは改めて自己紹介して頂いてよいですか？
・副業のご経験などはあるんですか？
・そっちの仕事は上手くいっているんですか？
・僕との出会いってそもそもどこからなんですか？
・弊社の求人を見て応募してくださったと思うんですけど、どんなところが気になって応募してくださったんですか？
・今後どういうキャリアを築いていきたいか、どうなっていきたいと感じているか、少し聞かせてもらえますか？
```

**深掘りフォロー:**
```
「ほんと自分の思い描いた未来を作っていけるし、ほんと誰でも実現できるんですよ。
未来は手に入るもので、わかっていると思うんですけど。じゃあ、これまでにその未来っていうのを
手にする為に何か行動されたことはありますか？」

「ちなみにどのようなものを取り組んできたんですか？」

「じゃあ、結構お金使いましたか？自分の未来を作ろうと思って起こした行動だと
思うんですけど、うまくいきました？」
```

## フェーズ3: 目的の明確化

```
「目指している先は素晴らしいと思います。なにかきっかけみたいのはあったんですか？」

「その築きを得れる感度があったのがすごいなと思うんですけど、じゃあ、なんで
[候補者名]さんはそれを実現しないといけないのか？というのを知りたいんですよ。
みなさんそれぞれの目的があると思うんですよ。なんでそれを実現したいんですか？」

「自分が実現することで始めてまわりの人たちに幸せの分配ができる人間になる、
それが、僕と接点を深めることで、今回だと面接に合格できれば実現出来ると思って
ここにきているということですね？」
```

## フェーズ4: コミットと目標設定

```
「[候補者名]さんの未来を実現していく為に僕と携わったら実現することはできます。
そのためのきっかけも環境もすべてお渡しします。」

「今まで[候補者名]さんから話を聞いてきてこのようになりたいとおっしゃっていて。
その目的を達成させる為の環境が必要ということですが、ただ僕は、[候補者名]さんと
この話を通じて[候補者名]さんが絶対にやったほうがいいな。と思ったら一緒にやるこになるし、
ただもし一緒にやるといったら全力で頑張ってもらわないといけないし、人生を変える気持ちが
あるのなら本気で行動に起こさないといけない訳ですよ。それは分かっていますか？」

「本気になれるんですね？なんでそこまで本気になれるんですか？」
```

**目標金額の設定:**
```
「具体的に、どれくらいの金額を目指していますか？」

「[目標金額]。今まで[目標金額]って稼いだことありますか？」

「1年以内に[目標金額]を絶対に実現する、それができたら最低限自分の目指すビジョン
というのは実現できる。ということですね？」
```

## フェーズ5: フルコミット確認

```
「なので、[候補者名]さんものすごく忙しいときもあるだろうし、これから今まで
やったことないことをやるわけです。それでも、その環境の中で迷子になりながら、
もちろん道しるべは作りますが一生懸命、ストイックに素直に言い訳せずに頑張れますか？」

「色んな壁が訪れてくるんですよ。たとえば、自分がやったこともないことだから
そのための勉強もしないといけない。勉強ができたら今度はそれを実践しないと形にできない。
でもどこか言い訳をして自分の物事の基準で動いてしまったら結局いつまでたっても
元通りの人間になってしまう。」

「だから、[候補者名]さんのその理念をちゃんと実現するために本気かどうか知りたいんですけど
本気ですか？なんでそこまで本気なんですか？」
```

**本気確認の連打:**
```
「本気ですか？」
「何の為ですか？」
「本気なんですよね？　めちゃめちゃ頑張れますか？」
「今までやったことないことをやっていく訳です。色んなことを吸収して勉強しないと
到達はできないかもしれない。でもそれをちゃんと手にできる環境はあるので、じゃあその中で
全力で自分の為に時にはいろんなことを犠牲にしないといけないかもしれませんが、
それを踏まえた上でも本気ってことですよね？」
```

## フェーズ6: テストクロージング

```
「じゃあ、合格です！」

（間を置いて感情を受け取らせてから）

「そこまでおっしゃるんだったらこっちも全力でやるんで一緒に頑張りましょう。」

「すごいなと思います。本気さというのはちゃんと伝わってきました。頑張りましょうね、
未来は明るいですよ。[候補者名]さんが望む未来というのはちゃんとやることをやれば、
当然ながら実現できるので、ちょっと厳しいこといいましたが、覚悟・気持ちを見たかったんですよ。」
```

## フェーズ7: クロージング（価格提示）

**入り口:**
```
「いちお今回の[商品名]というのも契約をしないといけないんですけど、
その話をこれからしていきます。」
```

**価格提示（比較から入る）:**
```
「[比較対象（競合・代替手段）]って、[比較価格]するんですよ。
それと同等以上の内容で、[商品の強み]です。
[商品名]は[価格]でご提供しています。それは大丈夫ですか。」
```

**ROIを示す:**
```
「[目標月収]稼げるようになれば、[価格]は[回収期間]で回収できます。
投資として見たとき、[比較対象]と比べてどっちが合理的かって考えると、
答えは出てくると思うんですけど。」
```

**責任・コミットのフレーム:**
```
「僕もリスクを抱えて事業行っていく際のリスクを持つわけですからリスクを抱えているし
みんなを成功させないといけない。責任を持っているわけです。だから僕もやることと
ちゃんとやるので、[候補者名]さんもしっかり覚悟を持って参加してもらいたいんですよ。」
```

**支払い方法の確認（優先順）:**
```
ネットバンク → 手渡し → クレカ の順番

「ちなみに、[候補者名]さんネットバンクってもっていますか？
ご用意出来来ましたら、振込先教えるんでそこにお支払いいただけますか？」
```

**締め:**
```
「[候補者名]さんね今すぐにやらないといけないことあると思うのでその気持ちというのを
閉ずることなく行動していけば絶対[目標]達成できるので頑張っていきましょう。
あとはこれからの未来を作っていくということをホントワクワクしてほしいなと思います。」

「ではこの後は契約書を交わさないといけないのでこの後説明させて頂きます。
今日はこれで面接は以上となります。」
```

## フェーズ8: 反論処理（汎用）

| 反論パターン | 返しのフレーム |
|------------|--------------|
| 「お金が今ない」 | 奨学金の喩え →「自己投資するためにお金がない人はどうするか？奨学金という形で借りてとかする訳じゃないですか。そのうえで[候補者名]さんどうしますか？」 |
| 「考えさせてほしい」 | 「今すぐやらないといけない気持ちを閉ざさないで。何が一番引っかかっていますか？」 |
| 「本当に稼げるか不安」 | 「動いた人だけが結果を手にする。実際にやった人の事例を見てもらって」 |
| 「家族に相談しないと」 | 「ご家族の収入が増える話ですよね。一緒に相談できる時間を作れますか？」 |
| 「もう少し後で」 | 「今の気持ちが一番高い。後になるほど動けなくなる人を沢山見てきています」 |

## 商品固有のカスタマイズ項目

新しい商品に適用するときは以下を差し替える：

```yaml
商品名: [商品名]
価格: [価格]
比較対象: [競合・代替手段]
比較価格: [競合の価格]
商品の強み: [この商品ならではの価値]
目標月収: [候補者が目標とする月収]
回収期間: [価格 / 目標月収 で計算した期間]
支払い方法: ネットバンク / 手渡し / クレカ
```
