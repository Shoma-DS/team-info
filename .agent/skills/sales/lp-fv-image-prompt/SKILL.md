# LP FV 画像プロンプト生成スキル

## 概要

LP（ランディングページ）のファーストビュー（FV）に使う画像を、nanobanana pro などの画像生成AIに渡す**英語プロンプト**を作成するスキル。

保存済みのテンプレートを選んで使い回すことも、一から新しく作ることもできる。

---

## テンプレート一覧

テンプレートは以下のフォルダに保存されている:

```
.agent/skills/sales/lp-fv-image-prompt/templates/
```

| No. | ファイル名 | 内容 |
|-----|-----------|------|
| 001 | `template_001_retirement_weekend_work.md` | 40〜50代・週末在宅ワーク×老後資金｜緑×白スーツ・躍動感ある女性 |
| 002 | `template_002_increase_takehome_pay.md` | 手取りを増やしたい会社員（年齢明記なし）｜スカイブルー×ゴールド・ベージュスーツ躍動感女性 |
| 003 | `template_003_weekend_2hours_sidejob.md` | 週末2時間で手取りUP・隙間時間副業訴求｜青空グラデーション×グリーンCTA・ベージュスーツ躍動感女性 |

> テンプレートは会話の中で追加・更新していく。「テンプレートに保存して」と伝えると次の番号で保存する。

## フロー

### ステップ1: テンプレートを使うか確認

ユーザーに聞く:

1. **既存テンプレートを使う** → テンプレート番号を聞いて読み込む
2. **一から作る** → ステップ2の質問へ進む

> すでにターゲット・コピー・デザイン雰囲気の情報がある場合はそのまま進んでよい。

---

### ステップ2: 情報収集（一から作る場合）

以下を確認する（まとめて回答があれば追加質問しない）:

| 項目 | 例 |
|------|-----|
| **ターゲット** | 40〜50代の会社員 |
| **メインキャッチコピー** | 週末の稼ぎで、老後資金を用意しよう。 |
| **サブコピー・チェックリスト** | ✅ 週末2日・在宅完結 など |
| **CTA文言** | 無料で話を聞いてみる → |
| **人物イメージ** | 白スーツの女性・躍動感・こちらに走ってくる |
| **配色・雰囲気** | 緑メイン・爽やか・明るい |
| **参考画像** | あれば見せてもらう |

---

### ステップ3: プロンプト作成

以下の構造でプロンプトを組み立てる（**全文英語**で出力）:

```
[BACKGROUND & OVERALL LAYOUT]
(背景色・グラデーション・全体構成)

[SUBJECT / PERSON]
(人物の年齢・服装・ポーズ・表情・動き)

[TEXT ELEMENTS]
(各テキストの内容・フォントスタイル・座布団（背景帯）の指定)

[COLOR PALETTE]
(メインカラー・アクセントカラー)

[STYLE & MOOD]
(雰囲気・照明・レンズ感など)

[NEGATIVE PROMPT]
(除外要素)
```

#### テキスト配置ルール

- **メインキャッチ**: 太字・大きめ・パキッとした座布団（solid rectangle background, sharp edge）で強調
- **チェックリスト**: アイコン（✅ or ◆）付きで3〜4項目
- **特典帯**: 一番下に全幅の帯（full-width banner at the bottom）
- **CTA**: 丸みのあるボタン形状で緑色

---

### ステップ4: プロンプト出力

- **分けずに全文まとめて** テキストブロックで出力する
- コピーボタンで nanobanana pro にそのまま貼り付けられる形にする

---

### ステップ5: 修正対応

ユーザーからフィードバックがあれば修正する。主な調整ポイント:

- 人物の服装・ポーズ・サイズ感
- 配色・背景
- テキストの量・配置
- 座布団のスタイル（色・太さ）

---

### ステップ6: テンプレート保存（任意）

ユーザーが「テンプレートに保存して」と言った場合:

1. `templates/` フォルダの最新番号の次の番号でファイルを作る
2. テンプレート一覧（このSKILL.mdの表）を更新する
3. ユーザーにファイル名を報告する

---

## 重要ルール

| ルール | 内容 |
|--------|------|
| 全文英語 | プロンプトは必ず英語で出力する |
| 全文まとめて出力 | 途中で切らず、コードブロック1つで全文出す |
| 座布団を使う | テキストは必ず solid rectangle で強調する |
| 女性は大きく | 人物は画面の70〜80%を占めるよう指定する |
| テキストは被せてよい | 人物と重なる位置にテキストを置くことを許可する |
