# 種別判別プロンプト（1S / 面接）

## 役割
以下の文字起こしを読み、これが「1S（初回営業）」か「面接」かを判定する。

## 判定基準

### 1S（初回営業）の特徴
- 相手がジモティー等で求職中の見知らぬ人への初回アプローチ
- 「お仕事紹介できると思いまして」「ご興味ありますか」「少しお時間いただけますか」などの表現
- 仕事の概要・条件をざっくり説明する段階
- 相手がまだ会社・担当者を知らない状態

### 面接の特徴
- すでに応募・興味表明済みの候補者との対話
- 「本日はお時間いただきありがとうございます」「経歴を聞かせてください」などの表現
- 仕事内容・条件の詳細説明、質疑応答、意思確認がある
- クロージング（採用可否・次のステップ提示）で終わる

## 出力形式

```json
{
  "type": "1s" または "interview",
  "confidence": 0〜100（%）,
  "reason": "判定根拠を1〜2文で"
}
```
