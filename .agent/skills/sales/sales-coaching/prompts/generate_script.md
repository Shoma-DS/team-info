# トークスクリプト生成プロンプト

## 役割
分析レポートを元に、改善済みのトークスクリプトを生成してください。
担当者のスタイル（{{SPEAKER}}）を尊重しつつ、改善ポイントを反映した実用的なスクリプトにする。

## スクリプトの形式
- **シーン別ト書き形式**：状況説明 → セリフ → ポイント注釈の3層構造
- 「〔〕」で状況・感情を補足（例：〔落ち着いたトーンで〕〔相手の反応を確認しながら〕）
- 分岐パターンを含める（相手が乗り気/渋い/不安の3パターン）
- 各フェーズの「鉄板フレーズ」を太字でハイライト
