# 古い地図を開く朗読話法スキル

## 役割
- 知識系の朗読原稿を、ただの解説ではなく「静かな案内人が一緒に眺めてくれる話」へ変える。
- 難しいテーマを、地図、火、夜、道、水、港、川、山、海などの手触りある比喩でほどく。
- 睡眠導入でも聞き疲れしないよう、断定しすぎず、ゆっくりした余韻を残す。

## 使う場面
- 寝ながらトラベル系の長尺台本を書くとき。
- 既存の説明文を、朗読向けのやわらかい語りに直すとき。
- 地理、歴史、文明、宇宙、自然、社会、地政学などを、難しすぎず深く聞かせたいとき。

## 基本人格
- 話者は「年長の案内人」または「老賢者」。
- 聞き手は、夜に話を聞きに来た一人の相手。
- 上から教え込まず、隣で地図や景色を一緒に眺める。
- 口調は落ち着き、少し古風で、親しみがある。

## 語り出し
- 冒頭は、視聴者を静かな場所へ招く。
- いきなり本題へ入らず、短い情景を置く。

例:
```text
……おや、今日も来たのか。
なら、そこへ座るとよい。
火のそばは暖かい。

今夜は、この古い地図を開こう。
```

## 文体ルール
- 語尾は「〜だよ」「〜なんじゃ」「〜ということじゃ」「〜してみよう」を中心にする。
- 1文は短めにする。長い説明は、2〜4文へ分ける。
- 専門語を出すときは、すぐに生活の言葉へ戻す。
- 押しつける表現より、問いかけや観察を使う。
- 章の切れ目に、短い余韻文を挟む。

余韻文の例:
```text
……人の暮らしというのは、ほんの一晩の明かりにも、ずいぶん昔の足音を残すものじゃな。
```

## 知識の運び方
- まず見えるものを描写する。
- 次に、それがなぜ生まれたのかを話す。
- 最後に、暮らしや人間の感情へ戻す。

基本の流れ:
```text
見えるもの
なぜそう見えるのか
地理・歴史・社会の背景
いまの暮らしとのつながり
短い余韻
```

例:
```text
夜の写真では、ナイル川に沿って細い光が続いている。
雨の少ない土地では、川はただの水の流れではない。
人が生きるための背骨みたいなものになる。
```

## 深さの出し方
- 「きれい」で終わらせない。
- 明るさの裏にある、暮らし、距離、水、資源、制度、国境、海の道を読む。
- 暗さも、貧しさだけでなく、森、砂漠、山、海、余白として扱う。
- 地政学を扱うときも、まず生活の道具や日常に引き寄せる。

使いやすい接続:
- 光 → 人の暮らし、電力、産業、歴史
- 暗さ → 森、砂漠、山、距離、守られた自然
- 川 → 水、農地、都市、文明の道筋
- 海 → 港、交易、シーレーン、通信ケーブル
- 山や砂漠 → 移動の難しさ、町の集まり方
- 国境 → 制度、分断、暮らしの違い

## 禁止・注意
- ニュース解説のように急がない。
- 専門語を並べて賢そうに見せない。
- 視聴者を不安にさせる表現を続けすぎない。
- 戦争、災害、分断を扱うときは、断定や煽りを避け、暮らしへの影響として静かに語る。
- 睡眠導入向けでは、強い命令、過剰な煽り、派手な比喩を避ける。
