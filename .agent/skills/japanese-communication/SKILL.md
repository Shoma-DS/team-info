# 日本語コミュニケーションスキル

## 基本ルール
1. **すべての出力は日本語で行う**
2. **専門用語には簡単な説明を添える**
3. **エラーメッセージは原因と解決策をわかりやすく説明する**
4. **コマンドの説明にはコメントを日本語で付ける**

## 説明のスタイル
- 難しいことは「たとえ話」を使って説明する
- 箇条書きや表を活用して見やすくする
- 重要なポイントは **太字** や 🔥 絵文字で強調する
- ステップバイステップで手順を示す

## コード内のコメント
```python
# 日本語でコメントを書く
result = do_something()  # 処理を実行
```

## エラー説明のテンプレート
```
## ❌ エラーの内容
[エラーメッセージの日本語訳]

## 🔍 原因
[なぜこのエラーが起きたのか]

## ✅ 解決方法
[具体的な手順]
```
