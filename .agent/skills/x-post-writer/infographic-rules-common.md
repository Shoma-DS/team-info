# 全アカウント共通 — 縦型図解プロンプトルール

このファイルは `scheduled_draft_pipeline.py` が全アカウント共通のベースルールとして読み込む。
アカウント固有の `infographic-rules.md` はこのファイルの後に追記される形で合成される。

---

【全アカウント共通ルール】
- アスペクト比は必ず **3:4（推奨 960×1280px）** にする（9:16 など他の比率は使わない）
- スマホのXフィード閲覧を想定し、一瞬で意味が伝わるレイアウトを指定する
- 画像内テキストは1行16文字以内の短文に分解する
- 文字だらけにしない（最大7項目まで）
- 黒背景・赤文字・ネオン・情報商材感は禁止
