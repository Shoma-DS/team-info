【縦型図解プロンプトルール】

- 推奨背景: 淡いブルー系（#EAF6FF / #F5FBFF / #DFF3FF）
- メインカラー: ブルー (#2563EB) + ネイビーテキスト (#0F172A)
- アクセント: シアン (#38BDF8) または イエロー (#FACC15) を1点のみ
- 構成: 上部に大見出し → 中央に3〜4ステップの白カードフロー → 下部に結論（大きく）
- キャラクター指定: 上部右側または右下に配置（図面積の20〜25%以内）
