# nanobanana pro 画像プロンプト作成スキル

## 目的

nanobanana pro に入力してすぐ使えるプロンプトを、1枚ごとに完結した形で日本語で書き出す。
共通仕様の参照は使わず、各プロンプトを読めばすべてわかる自己完結型にする。

---

## プロンプトの構成テンプレート

AirthMate スライドプロンプトのフォーマットを基本テンプレートとして使う。
以下の構成を毎回すべて書く（省略・参照なし）。

```text
あなたは【トップクラスの○○デザイナー兼○○デザイナー】です。

@img と @logo を参考にしながら、[何のための画像か・一言で]。

目的：
[この画像が達成すべきことを1〜2行で]

必須条件：
[デザイン仕様を箇条書きで。以下の項目を毎回すべて書く]
- 画像サイズ・比率
- 背景色・グラデーション
- テキスト色
- 強調・アクセントカラー
- 角丸・直角の指定
- テキスト量の方針
- @img の扱い方
- @logo の扱い方
- 出典・数字の扱い方（必要な場合）
- 【禁止】ロケット・宇宙・UFO・AI回路・脳みそ・チップ系のアイコン・イラストは一切使わない
- 【禁止】絵文字（✅ ⭐ 🚀 💡 など）は使わない
- 【禁止】吹き出し・手書き風・マンガ的な表現は使わない
- 【禁止】フリー素材っぽい安いクリップアート・イラストは使わない
- 【ビジュアルトーン】[このプロジェクトのトーンを一言で]
- 【人物を使う場合】[人物の使い方の指定]

[コンテンツ固有の情報]
タイトル：（ある場合）
メインコピー：
サブコピー：
その他テキスト：
レイアウト：
トーン：
```

## プロジェクト別・必須条件の定型文

### AirthMate スライド（16:9・ブルー系）

```text
- 16:9 の営業スライド
- コピーはすべて日本語
- ブルーグラデーション背景（`#3B4BCE` → `#5BBACF`）+ テキスト白 `#FFFFFF`
- 装飾的な幾何学シェイプ（半透明ブルー系）で奥行きを出す
- 強調・アクセントカラーは白または `#5BBACF`（明るいティールブルー）
- グリーン・ゴールド・ダーク系は使わない
- 角丸よりも直角。清潔感より「信頼感」と「意思の強さ」を優先
- テキスト量は多すぎない。1スライド 1メッセージ
- 見出しは極太、補足は短く、スマホでも読める密度
- `@img` の構造や余白感は参考にしてよいが、コピー・トレース・再現は禁止
- `@logo` がある場合は参照し、左上または四隅のどこかに小さく上品に配置する
- 数字や出典を扱う場合は、フッターに小さく出典を入れ、数字の信頼感を最優先する
- 【禁止】ロケット・宇宙・UFO・AI回路・脳みそ・チップ系のアイコン・イラストは一切使わない
- 【禁止】絵文字（✅ ⭐ 🚀 💡 など）は使わない
- 【禁止】吹き出し・手書き風・マンガ的な表現は使わない
- 【禁止】フリー素材っぽい安いクリップアート・イラストは使わない
- 【ビジュアルトーン】クリーンなビジネス資料。BtoB感・事業感・オフィシャル感を最優先する
- 【人物を使う場合】スーツ姿のフレッシュなビジネスパーソン（20〜40代）を @img から参照。人物は信頼感・臨場感の補強として配置し、主役はコピーと構造にする
```

### AirthMate LINE バナー（1:1・グリーン系）

スライドと同じ構造・同じプロンプト形式を使う。
カラーだけ LP に合わせてグリーン系に変える。

```text
- 1:1 の正方形画像（1040×1040px）。LINEのボタンテンプレートまたはカルーセルに使用する
- コピーはすべて日本語
- 背景は白（`#FFFFFF`）または薄グリーン（`#f0fdf4`）
- テキストはほぼ黒（`#1D2130`）またはダークグレー（`#4A5568`）
- 強調・アクセントカラーはグリーン（`#22C55E`）
- ゴールド（`#F59E0B`）はバッジ・実績数字にのみ使用可
- 黒座布団要素（`#111827`）はラベル・タグにのみ使用可
- 角丸よりも直角。カードや枠は直角（border-radius: 0）
- CTAボタンはpill型（横長・角丸9999px）。グリーン背景・白文字
- テキスト量は多すぎない。1画像 1メッセージ
- 見出しは極太、補足は短く、スマホでも読める密度
- `@img` の構造や余白感は参考にしてよいが、コピー・トレース・再現は禁止
- `@logo` がある場合は参照し、目立たない位置に小さく上品に配置する
- 【禁止】ロケット・宇宙・UFO・AI回路・脳みそ・チップ系のアイコン・イラストは一切使わない
- 【禁止】絵文字（✅ ⭐ 🚀 💡 など）は使わない
- 【禁止】吹き出し・手書き風・マンガ的な表現は使わない
- 【禁止】フリー素材っぽい安いクリップアート・イラストは使わない
- 【ビジュアルトーン】フレッシュでクリーンなコーポレート。LP・スライドと統一された世界観。情報商材感ゼロ
- 【人物を使う場合】スーツまたはカジュアルなビジネス服装のフレッシュな人物（20〜40代）を @img から参照
```

### AirthMate LINE チャット風感想画像（1:1・スマホUI風）

```text
- 1:1 の正方形画像（1040×1040px）。LINEのカルーセルカードに使用する
- コピーはすべて日本語
- 背景は薄いグレー（`#F2F2F7`）。スマートフォンのトーク画面の雰囲気
- LINEの正式なロゴ・UIは使わない。「LINEっぽい雰囲気」のオリジナルデザインにする
- メッセージバブルは白・角丸・左寄せ（受信メッセージスタイル）
- 上部ヘッダーは白背景・中央にアイコンと「ChatGPT活用実践セミナー」のテキスト
- タイムスタンプはグレー・小文字
- 下部タグは黒座布団（`#111827`）pill型・白文字で参加者の属性を表示
- テキストは読みやすく・バブル内に収まる量に抑える
- `@img` の構造や余白感は参考にしてよいが、コピー・トレース・再現は禁止
- 【禁止】本物のLINEのロゴ・UI要素を使わない
- 【禁止】実在する人物の顔写真は使わない。アイコンはシルエットまたは抽象的なもの
- 【禁止】ロケット・宇宙・UFO・AI回路・脳みそ・チップ系のアイコン・イラストは一切使わない
- 【禁止】絵文字（✅ ⭐ 🚀 💡 など）は使わない
- 【ビジュアルトーン】リアルで信頼感のある口コミ風。清潔感・自然さを最優先する
```

## 出力形式

各画像は以下のブロック形式で出力する。```text ... ``` の中身をそのままコピペして nanobanana pro に貼る。

````markdown
---

### 【画像①】画像の名前

```text
（ここにプロンプト本文。上記テンプレートに従って完全に書く）
```

---
````

## 注意事項

- プロンプトは日本語で書く
- 1枚1枚完全に独立して書く。「前の画像と同じ」「共通仕様参照」は禁止
- 必須条件は省略せず毎回すべて書く
- テキストの内容は実際に画像に入れる文言をそのまま記載する
- レイアウトは上から順番に要素を記述する
- nanobanana pro は役割定義から始めること（「あなたは【...】です。」）
