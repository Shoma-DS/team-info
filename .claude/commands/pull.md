リモートの最新状態を手元に取り込みます。

## 手順

1. `git -C "$TEAM_INFO_ROOT" fetch origin` でリモートの最新情報を取得する
2. `git -C "$TEAM_INFO_ROOT" status` で現在の状態を確認する
3. 未コミットの変更がある場合は、先にコミットするかどうかユーザーに確認する
4. `git -C "$TEAM_INFO_ROOT" pull --rebase origin main` で最新状態に更新する
5. コンフリクトが発生した場合は、ユーザーに手動解決を依頼する
6. 完了後、更新されたブランチ名と取り込んだコミット数をユーザーに報告する

## 報告形式

```
✅ 最新状態に更新しました

ブランチ: main
取り込んだコミット: X件（または「すでに最新です」）
```
