これは Claude Code 用の互換ラッパーです。正本は `AGENTS.md` の開発モード管理ルールです。

以下の手順で個人開発モードに切り替えてください。

## 手順

1. `.dev-mode` ファイルを読み込み、現在のモードを確認する
2. `personal` と書き込んでモードを切り替える
3. ユーザーに以下の形式で報告する：

```
🔒 個人開発モードに切り替えました

【個人開発モード】
- 新規作成するファイル・フォルダは自動的に .gitignore に追加されます
- チームには共有されません（git の追跡対象外）
- 既存のファイルへの変更はこのルールの対象外です
```

## 注意
- .dev-mode ファイル自体は .gitignore に含めること（個人設定のため）
