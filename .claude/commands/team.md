以下の手順でチーム開発モードに切り替えてください。

## 手順

1. `/Users/deguchishouma/team-info/.dev-mode` ファイルを読み込み、現在のモードを確認する
2. `team` と書き込んでモードを切り替える
3. ユーザーに以下の形式で報告する：

```
✅ チーム開発モードに切り替えました

【チーム開発モード】
- 新規作成するファイル・フォルダは通常どおり git の追跡対象になります
- コミット・プッシュすることでチームと共有されます
- .gitignore への自動追加は行いません
```

## 注意
- .dev-mode ファイル自体は .gitignore に含めること（個人設定のため）
- 既存の .gitignore は変更しない
