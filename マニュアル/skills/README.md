# スキルマニュアルショートカット

このフォルダには、各スキルへのショートカット（主にシンボリックリンク）を置きます。

## 命名ルール
- `*_manual.md`: 利用者向けマニュアル（`MANUAL.md`）
- `*_skill.md`: スキル定義（`SKILL.md`）
