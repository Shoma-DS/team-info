# マニュアル集

このフォルダは、プロジェクト内のマニュアルを集約して参照しやすくするための入口です。

- まず読む概要マニュアル: `マニュアル/まずはこちらをお読みください.md`
- Discord Webhook の作り方: `マニュアル/Discord Webhook設定手順.md`
- スキル別ショートカット: `マニュアル/skills/`
- 正本のルール:
  - `AGENTS.md`
  - `RULES.md`
  - 各スキルの `SKILL.md`

## 使い方
- team-info 全体像を確認したいときは `マニュアル/まずはこちらをお読みください.md` を参照する。
- スキルの入口だけ素早く見たいときは `マニュアル/skills/` を参照する。
- 細かい運用ルールや最新の作法は、必ず `AGENTS.md` と `RULES.md` を見て確認する。
