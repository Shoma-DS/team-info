# スラッシュコマンド一覧

チャットに `/コマンド名` と入力するだけで、AI が動いてくれます。

---

## モード切り替え（最初に設定する）

作業が「チームに共有するもの」か「自分だけのもの」かを決めます。

| コマンド | 何をするか |
|---------|-----------|
| `/team` | チームモードにする。作ったファイルはチームと共有される |
| `/personal` | 個人モードにする。作ったファイルは自分だけに残り、チームには見えない |

> 今どっちのモードか分からなくなったら `/team` か `/personal` を入力するだけでOK。

---

## 開発・制作のコマンド

| コマンド | 何をするか |
|---------|-----------|
| `/pull` | チームの最新の変更を自分のパソコンに取り込む（作業前に必ず使う） |
| `/c` | 今の変更を保存する（チームには送らない。こまめに使う） |
| `/git` | 変更をチームに送る（作業が終わったときに使う） |
| `/script` | 動画の台本を作る |
| `/voice` | 音声の読み上げテキストを作る |
| `/lyric` | 歌詞動画を作る |
| `/sleep-travel` | 「寝ながらトラベル」の動画を作る |
| `/acoriel` | 「アコリエル」のアコースティックカバー動画を作る |
| `/jmty` | ジモティーの投稿文を作る |

---

## 便利なコマンド

| コマンド | 何をするか |
|---------|-----------|
| `/setup` | このリポジトリ（チームの共有フォルダ）を最初に準備する |
| `/gdrive` | Google ドライブにファイルをコピーする |
| `/tyoudoii-illust-fetcher` | ちょうどいいイラストを探してダウンロードする |
| `/themeisle-illustration-fetcher` | Themeisle のイラストを探してダウンロードする |
