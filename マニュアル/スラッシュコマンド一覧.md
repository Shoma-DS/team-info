# スラッシュコマンド一覧

Claude・Codex・Gemini などの AI エージェントに対して使えるコマンドです。
チャットで `/コマンド名` と入力するだけで動作します。

---

## 開発モード切り替え

| コマンド | 説明 |
|---------|------|
| `/team` | **チーム開発モード**に切り替え。新規ファイルは通常どおり git で共有される |
| `/personal` | **個人開発モード**に切り替え。新規ファイル・フォルダは自動で `.gitignore` に追加され、git の追跡対象外になる |

> 現在のモードは `.dev-mode` ファイルで管理されています。
> エージェントはファイル作成を伴うタスクの前に、必ずモードを確認してから作業を進めます。

---

## 開発・制作スキル

| コマンド | 説明 |
|---------|------|
| `/pull` | リモートの最新状態を取り込む（作業開始前に実行） |
| `/c` | **コミットのみ**（push・PR なし）。オーナー機は main、それ以外はアカウント名のブランチにコミット |
| `/git` | 変更内容を確認してコミット＋プッシュ（オーナー機以外は PR 作成） |
| `/script` | 動画台本を作成（チャンネル対応） |
| `/voice` | 音声スクリプトを生成・VOICEVOXで読み上げ |
| `/lyric` | 歌詞動画の感情マッピングと制作 |
| `/sleep-travel` | 寝ながらトラベルの動画制作 |
| `/acoriel` | アコリエルのアコースティックカバー動画制作 |
| `/jmty` | ジモティー投稿文を作成 |
